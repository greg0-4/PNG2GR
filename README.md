# Simple python script converting PNG to GR file
Here is proof of it working

![Image](proofOfWorking.png)

# Dependencies 
Pillow library in python
<br>
Python 3.14.4 (i dont think it matters, just writing it down to be sure)

# How to use
You go into the png2gr.py file and edit the function argument with
the image path and it will give you an "output.GR" file which is your converted image

# Notice
This does not support any colors outside of the TempleOS color palette.
